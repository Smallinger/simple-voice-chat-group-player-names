A NeoForge port of the popular Simple Voice Chat Group Player Names mod that displays player names next to group player heads in the [Simple Voice Chat](https://modrinth.com/mod/simple-voice-chat) HUD.

## âš™ï¸ Configuration

Access the configuration through the Mods menu:
1. Click on **Mods** in the main menu
2. Find **Simple Voice Chat Group Player Names**
3. Click the **Config** button

### Available Options:

| Option | Description | Default |
|--------|-------------|---------|
| **Only show names when talking** | Display player names only when they are speaking | `false` |
| **Use text outline** | Draw a dark outline around names for better visibility | `false` |
| **Transparency when talking** | Transparency level (0-100%) when player is talking | `100%` |
| **Transparency when not talking** | Transparency level (0-100%) when player is silent | `50%` |
